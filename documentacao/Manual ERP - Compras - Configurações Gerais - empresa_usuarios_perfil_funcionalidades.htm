Manual ERP

Configurações Gerais

Este espaço é destinado a manipular configurações de usuários, produto, financeiro, faturamento e sistema.

O administrador do sistema poderá inserir/excluir usuários, criar/excluir categorias de produto e unidades de medida, criar/excluir centro de custos de financeiro, criar/excluir serie de notas fiscais em faturamento, alterar opções de estoque e inventário no sistema.

 

 

Usuários

Para estabelecer os diferentes usuários do sistema, é necessário predefinir os níveis de acesso e funções de cada patamar para permitir visualizar/manipular as informações segundo a necessidade de cada um deles.

Dentro de Usuários, é possível:

·         Criar Usuários

·         Excluir Usuários

Para criar um novo usuario, selecione a aba “Usuários” e depois clique em “Novo Usuario”.

 

Novo Usuário

Para cadastrar um novo usuário são necessárias as seguintes informações:

Nome

Email

Senha

Confirmar senha

Selecione o perfil desejado: para cada perfil de usuário desejado, possuirá acesso às diferentes opções conforme descrito abaixo:

·         Administrador: Terá acesso total a todos os níveis.

·         Comprador: Terá acesso aos cadastros e preço de custo, juntamente com funções de compra e estoque.

·         Vendedor sênior: Terá acesso a todas as opções de vendas, cadastros e notas fiscais.

·         Vendedor júnior: Terá acesso à opção de visualização da sua própria venda e visualiza e edita cadastro no momento da venda.

·         Financeiro sênior: Terá acesso a todas as opções de financeiro e relatórios vinculados ao financeiro.

·         Financeiro júnior: Terá acesso a todas as opções do financeiro.

 

Após preencher as informações, clique em “Salvar”.

Excluir

Para excluir um cliente, selecione o cliente e clique no botão Excluir.

 

 

Produto

Nesta configuração podem ser inseridos itens customizados de Categoria de Produtos e também Unidades de Medida conforme a necessidade requisitadas pela empresa.

Dentro de Produto, é possível:

·         Categoria de Produto

o   Criar nova categoria

o   Excluir categoria

·         Unidades de medida

o   Criar nova unidade de medida

o   Excluir unidade de medida

Categoria de Produto

Para criar uma nova categoria, clique em “Nova Categoria”.

 

 

 

Nova categoria

Para cadastrar uma nova categoria, só será necessário inserir o nome da categoria.

Após preencher a informação, clique em “Salvar”.

Excluir

Para excluir uma categoria, selecione a categoria e clique no botão Excluir.

 

Unidades de medida

Para criar uma nova unidade de medida, clique em “Nova Unidade de Medida”.

 

Nova Unidade de Medida

Para cadastrar uma nova unidade de medida, são necessárias as seguintes informações:

Unidade de medida

Símbolo

Deixe marcada a caixa de seleção “Ativo” para deixar a nova unidade operante, ou caso contrario, desmarque-a.

 Após preencher a informação, clique em “Salvar”.

Excluir

Para excluir uma unidade de medida, selecione a unidade de medida e clique no botão Excluir.

 

 

 

Financeiro

Para permitir uma melhor identificação da unidade geradora de despesa e/ou receita tem a opção de gerar o centro de custo.

Dentro de Financeiro, é possível:

·         Criar Centro de Custo

·         Excluir Centro de Custo

Para criar um novo Centro de Custo, selecione a aba “Financeira” e depois clique em “Novo Centro de Custo”.

 

 

Novo Centro de Custo

Para cadastrar um novo centro de custo, só será necessário inserir o nome do centro de custo.

Após preencher as informações, clique em “Salvar”.

Excluir

Para excluir um centro de custo, selecione o centro de custo e clique no botão Excluir.

 

 

Faturamento

Para facilitar o controle das operações fiscais de entrada ou saída realizadas pela empresa, deve ser utilizado o numero de serie da nota fiscal.

Dentro de Faturamento, é possível:

Informações da serie da nota fiscal

Serie

Serie para venda (3 tipos):

o   Produto

o   Serviço

o   Ambos

Numero atual

 

Após preencher a informação, clique em “Salvar”.

Excluir

Para excluir uma categoria, selecione a categoria e clique no botão Excluir.

 

 

Sistema

O sistema possibilita que o produto cadastrado contenha as opções de validação de estoque e as diferentes naturezas das operações do inventário para notas fiscais.

 

Dentro de Sistema, é possível a escolha dos seguintes filtros:

·         Estoque

o   Validar Estoque ao emitir uma venda

o   Validar Estoque ao emitir pedido de venda

·         Inventário

o   Natureza de operação para NF’s de entrada, com as seguintes opções:

ü  Compra

ü  Devolução de Compra

ü  Devolução de Venda

ü  Remessa de Produto

ü  Retorno de Remessa

ü  Venda

ü  Venda – cupom fiscal a prazo

ü  Venda – cupom fiscal a vista

o   Natureza de operação para NF’s de saida, com as seguintes opções:

ü  Compra

ü  Devolução de Compra

ü  Devolução de Venda

ü  Remessa de Produto

ü  Retorno de Remessa

ü  Venda

ü  Venda – cupom fiscal a prazo

ü  Venda – cupom fiscal a vista

 

Após escolher a informação, clique em “Salvar”.

Excluir

Para excluir uma categoria, selecione a categoria e clique no botão Excluir.
